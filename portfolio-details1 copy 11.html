Description

This project is a React-based e-commerce application that allows users to explore and filter products by categories, subcategories, and price ranges. Key features include: Dynamic Filtering: Users can search, filter by price, and select subcategories. Product Listings: Products are displayed with images, descriptions, and prices. Add to Cart: Easily add items to the cart with instant feedback using toast notifications. Navigation: Users can view detailed product pages. SEO-Friendly: Helmet integration ensures optimized meta tags for better visibility. The project integrates with a backend API for data fetching and offers a smooth, user-friendly interface.
